## Overview
Static portfolio website hosted on Azure Blob Storage without provisioning any server.

## Live URL
https://yomnaportfolio2026.z28.web.core.windows.net/

## Files
- `index.html` — Main portfolio page
- `404.html` — Custom error page

## Tools Used
- Azure Blob Storage (Static Website Hosting)
- Azure Storage Account (Standard LRS, France Central)

## Steps Taken
1. Created Azure Storage Account
2. Enabled Static Website hosting → set index.html + 404.html
3. Uploaded files to $web container
4. Accessed live public URL
